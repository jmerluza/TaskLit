# TaskLit
A side project to make good old task scheduler more lit to use. 

## Nice to have features
1. Create basic task.
2. Import task.
3. Delete task.
4. Run Task.
5. Search for task.
6. Monitor tasks for each folder of users choosing:
  - name
  - description
  - next_run_time
  - last_run_time
  - last_task_result
  - missed_runs
  - author
7. Dependency
